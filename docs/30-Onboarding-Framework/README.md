## 🏛️ Executive Summary
This onboarding framework provides standardized processes, documentation guidelines, and templates to scale the onboarding of Taluka Heads (THs) and Field Sales Executives (FSEs) across India.

## 📂 Framework Directory Structure
- [01-Onboarding-Overview.md](01-Onboarding-Overview.md): Overview of role definitions and lifecycle goals.
- [02-Onboarding-Workflow.md](02-Onboarding-Workflow.md): Detailed 16-stage pipeline flowchart.
- **03-Taluka-Head/**: Candidate forms, legal agreements, KYC checks, NDAs, and first-month plans for TH partners.
- **04-Field-Sales-Executive/**: HR files, salary sheets, incentive frameworks, and onboarding checks for FSE staff.
- **05-Shared/**: Corporate profiles, manuals, catalogues, playbook logs, and escalation pathways.
- **06-Templates/**: Copy-pasteable emails, WhatsApp formats, MoM, and review scorecards.
