# School Furniture Technical Specifications

This specification sheet defines the technical and quality standards for classroom student desks and benches procured via the DnyanMitra platform.

---

## 📐 Dual Desk (Student Bench) Specifications

All bidding vendors must ensure their products match or exceed the parameters listed below. Mismatched dimensions will result in bid disqualification.

| Attribute | Requirement |
| :--- | :--- |
| **Product Category** | Secondary Classroom Dual Desk |
| **Overall Dimensions** | 1050mm (Width) x 900mm (Depth) x 750mm (Desk Height) |
| **Seat Height** | 450mm |
| **Frame Material** | 25mm x 25mm CRCA square pipe, minimum thickness 1.2mm |
| **Finish** | Epoxy polyester powder coated (minimum 50 microns thickness) |
| **Wood Component** | 18mm pre-laminated commercial particle board with PVC edge banding |
| **Book Shelf** | Under-desk metal wire grid shelf (minimum 4.0mm wire gauge) |
| **Target Certification** | ISO 9001:2015, SEFA-8M compliant |

---

## 🛠️ Performance Test Checklist for Delivery

Taluka Heads and school store officers must execute these physical checks upon batch delivery:

1. **Verify Dimensions**: Use a metric tape measure to confirm width, depth, and height variations are under ±5.0mm.
2. **Edge Band Check**: Ensure the PVC edge banding on particle boards is sealed cleanly without visible glue leaks or peeling edges.
3. **Wobble Test**: Place the desk on a level concrete surface. Apply a 50kg downward load to verify frame alignment and stability.
4. **Powder Coat Scratch Check**: Visually inspect metal legs for paint chipping or exposed steel.
