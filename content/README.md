# Content Directory

This directory contains the actual business documents, presentations, guides, and outreach copy generated for the DnyanMitra and DASP Digital platforms.

Unlike the `/content-standards` directory which governs the *standards* and *templates*, this folder contains the *active materials* used in day-to-day operations.

## 📂 Subdirectories

- [presentations/](presentations/README.md): Slide decks, pitch scripts, and engagement guides for demo sessions.
- [training/](training/README.md): Operational training guides and onboarding manuals.
